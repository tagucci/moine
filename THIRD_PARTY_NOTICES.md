`moine` source code is licensed under either MIT or Apache-2.0. Dictionary data
is distributed separately from the source package and carries its own license
and attribution metadata.

## Dictionary Artifacts

- Japanese artifacts are derived from UniDic-CWJ. Published bundles must include
  the applicable UniDic license files referenced from `metadata.yaml`.
- Chinese artifacts are derived from CC-CEDICT. Published bundles must include
  CC-CEDICT attribution, source metadata, and CC BY-SA license information
  separately from the `moine` source-code license.

The base Python and Rust packages do not bundle UniDic-CWJ, CC-CEDICT, or
generated dictionary payloads.
